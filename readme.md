# Recipe-Discovery

## Descriere
Aplicația permite utilizatorilor să introducă ingredientele pe care le dețin pentru a le recomanda ce pot găti sau de ce ar mai avea nevoie pentru a găti diferite feluri de mâncare în funcție de dieta aleasă.

## Cerințe și specificații
- Adăugarea ingredientelor
- Afișarea rețetelor
- Afișarea ingredientelor de care mai are nevoie utilizatorul în funcție de rețetă
- Filtrarea rețetelor în funcție de ce dietă vrea să urmeze utilizatorul
- Interfață simplă și intuitivă
- Stocarea datelor într-o bază de date
